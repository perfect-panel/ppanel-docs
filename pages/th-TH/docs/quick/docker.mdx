# คู่มือการติดตั้งอย่างรวดเร็ว

> **ข้อควรระวังก่อนการติดตั้ง**
>
> - **พอร์ตเริ่มต้นของการจัดการคือ 8080**: กรุณากำหนดการตั้งค่าการชี้โดเมนล่วงหน้าหรือให้แน่ใจว่ามีที่อยู่ IP ที่ใช้งานได้ เพื่อให้การติดตั้งส่วนการจัดการและส่วนผู้ใช้เป็นไปอย่างราบรื่น
> - เซิร์ฟเวอร์: สำหรับข้อมูลเพิ่มเติมโปรดดูที่ [เซิร์ฟเวอร์](/th-TH/docs/server/config)
> - ส่วนการจัดการ: สำหรับข้อมูลเพิ่มเติมโปรดดูที่ [ส่วนการจัดการ](/th-TH/docs/admin/config)
> - ส่วนผู้ใช้: สำหรับข้อมูลเพิ่มเติมโปรดดูที่ [ส่วนผู้ใช้](/th-TH/docs/user/config)

## การติดตั้งด้วยคำสั่งเดียว

ใช้คำสั่งใดคำสั่งหนึ่งด้านล่างนี้เพื่อทำการติดตั้ง PPanel โดยรวมถึงเซิร์ฟเวอร์, ส่วนการจัดการ และส่วนผู้ใช้:

### ตัวเลือก 1: ใช้ `curl`

```bash
bash <(curl -fsSL https://raw.githubusercontent.com/perfect-panel/ppanel-script/refs/heads/main/install.sh)
```

### ตัวเลือก 2: ใช้ `wget`

```bash
bash <(wget -qO- https://raw.githubusercontent.com/perfect-panel/ppanel-script/refs/heads/main/install.sh)
```
