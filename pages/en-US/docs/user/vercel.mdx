# Deploy the user app with Vercel

The user app is in `apps/user` of the current
[frontend](https://github.com/perfect-panel/frontend) repository. Select that
directory as the Vercel Root Directory and set at least:

```text
VITE_API_BASE_URL=https://api.example.com
```

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https%3A%2F%2Fgithub.com%2Fperfect-panel%2Ffrontend&root-directory=apps%2Fuser&project-name=ppanel-user-web)

Redeploy after changing environment variables. The old `ppanel-web` deployment
link does not apply to the current frontend repository.
