# Build the user app from source

The current user app is a Vite application. The fixed `v1.0.0` download from
the former `ppanel-web` project is no longer provided.

```bash
git clone https://github.com/perfect-panel/frontend.git
cd frontend
cp apps/user/.env.example apps/user/.env.production
# Edit apps/user/.env.production; at minimum set VITE_API_BASE_URL
bun install --frozen-lockfile
bun run build --filter=ppanel-user-web
```

Deploy `apps/user/dist/` to a static web server, object storage, or CDN.
Node.js/Bun is only needed for the build.

See the [frontend deployment guide](https://github.com/perfect-panel/frontend/blob/main/docs/guide/separation/frontend.md)
for complete deployment options.
