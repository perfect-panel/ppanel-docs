# Host the user app with Docker

The user app is a Vite static site. Configure `apps/user/.env.production` as
described in [Environment variables](./config), then build it:

```bash
git clone https://github.com/perfect-panel/frontend.git
cd frontend
bun install --frozen-lockfile
bun run build --filter=ppanel-user-web
```

Serve the static files with an Nginx container:

```bash
docker run -d --name ppanel-user-web --restart unless-stopped \
  -p 3002:80 \
  -v "$(pwd)/apps/user/dist:/usr/share/nginx/html:ro" \
  nginx:alpine
```

Configure HTTPS for the public domain. `VITE_API_BASE_URL` must point to a
backend API URL that browsers can reach.
