# Admin environment variables

The admin application lives in `apps/admin` of
[perfect-panel/frontend](https://github.com/perfect-panel/frontend) and is
built with Vite. The old `NEXT_PUBLIC_*` variables and
`perfect-panel/ppanel-web` links no longer apply.

Set the following in `apps/admin/.env.production`:

```dotenv
# Required: public backend API URL
VITE_API_BASE_URL=https://api.example.com

# Optional: API path prefix provided by a reverse proxy
VITE_API_PREFIX=

# Optional: static asset CDN
VITE_CDN_URL=https://cdn.example.com

# Optional: show the tutorial entry; defaults to true
VITE_TUTORIAL_DOCUMENT=true

# Development only; keep empty in production
VITE_USER_EMAIL=
VITE_USER_PASSWORD=
```

Vite embeds these values into the static build. Rebuild and redeploy the admin
application after changing the API or CDN URL. See the current
[`.env.example`](https://github.com/perfect-panel/frontend/blob/main/apps/admin/.env.example).
