# Build the admin app from source

The current admin app is a Vite application. The fixed `v1.0.0` download from
the former `ppanel-web` project is no longer provided. Build static files from
the current frontend repository:

```bash
git clone https://github.com/perfect-panel/frontend.git
cd frontend
cp apps/admin/.env.example apps/admin/.env.production
# Edit apps/admin/.env.production; at minimum set VITE_API_BASE_URL
bun install --frozen-lockfile
bun run build --filter=ppanel-admin-web
```

Deploy `apps/admin/dist/` to any static web server, object storage, or CDN.
Node.js/Bun is only needed for the build, not at runtime.

See the [frontend deployment guide](https://github.com/perfect-panel/frontend/blob/main/docs/guide/separation/frontend.md)
for Nginx, Vercel, Netlify, and Cloudflare Pages options.
