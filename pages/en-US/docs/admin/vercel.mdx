# Deploy the admin app with Vercel

The admin app is in `apps/admin` of the current
[frontend](https://github.com/perfect-panel/frontend) repository. Select that
directory as the Vercel Root Directory and set:

```text
VITE_API_BASE_URL=https://api.example.com
VITE_CDN_URL=https://cdn.example.com
```

Use the prefilled deployment link for the official repository:

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https%3A%2F%2Fgithub.com%2Fperfect-panel%2Ffrontend&root-directory=apps%2Fadmin&project-name=ppanel-admin-web)

Environment variables are embedded at build time. Trigger a new deployment
after changing them; do not use the old `ppanel-web` Vercel link.
