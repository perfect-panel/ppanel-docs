# Host the admin app with Docker

The admin app is a Vite static site; it does not need a long-running Node.js
container. Configure `apps/admin/.env.production` as described in
[Environment variables](./config), then build it in the frontend repository:

```bash
git clone https://github.com/perfect-panel/frontend.git
cd frontend
bun install --frozen-lockfile
bun run build --filter=ppanel-admin-web
```

Serve the generated files with an Nginx container:

```bash
docker run -d --name ppanel-admin-web --restart unless-stopped \
  -p 3001:80 \
  -v "$(pwd)/apps/admin/dist:/usr/share/nginx/html:ro" \
  nginx:alpine
```

Use a reverse proxy for HTTPS in production and restrict access to the admin
application.
