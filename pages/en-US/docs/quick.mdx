# Quick Deployment Guide

> **Pre-Deployment Notes**
> - **Admin Panel Default Port is 8080**: Please configure domain resolution in advance or ensure an available IP address for a smooth installation of the admin and user panels.
> - Server: For more information, please refer to [Server](/en-US/docs/server).
> - Admin Panel: For more information, please refer to [Admin Panel](/en-US/docs/admin).
> - User Panel: For more information, please refer to [User Panel](/en-US/docs/user).

## One-Click Deployment

Use one of the following commands to deploy PPanel in one click, including the server, admin panel, and user panel:

### Option 1: Using `curl`

```bash
bash <(curl -fsSL https://raw.githubusercontent.com/perfect-panel/ppanel-script/refs/heads/main/install.sh)
```

### Option 2: Using `wget`

```bash
bash <(wget -qO- https://raw.githubusercontent.com/perfect-panel/ppanel-script/refs/heads/main/install.sh)
```
