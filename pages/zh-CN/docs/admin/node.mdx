# 从源码部署管理端

管理端当前是 Vite 应用，不再提供旧版 `ppanel-web` 的固定 `v1.0.0` 下载包。
使用当前前端仓库构建静态文件：

```bash
git clone https://github.com/perfect-panel/frontend.git
cd frontend
cp apps/admin/.env.example apps/admin/.env.production
# 编辑 apps/admin/.env.production，至少设置 VITE_API_BASE_URL
bun install --frozen-lockfile
bun run build --filter=ppanel-admin-web
```

将 `apps/admin/dist/` 部署到任意静态 Web 服务器、对象存储或 CDN。运行环境不需要
Node.js；Node.js/Bun 只用于构建阶段。

完整的 Nginx、Vercel、Netlify 与 Cloudflare Pages 方案见
[前端部署指南](https://github.com/perfect-panel/frontend/blob/main/docs/zh/guide/separation/frontend.md)。
