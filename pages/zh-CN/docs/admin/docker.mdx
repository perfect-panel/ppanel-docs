# 使用 Docker 托管管理端

管理端是 Vite 静态站点，没有需要长期运行的 Node.js 容器。先按[环境变量说明](./config)
配置 `apps/admin/.env.production`，再在前端仓库中构建：

```bash
git clone https://github.com/perfect-panel/frontend.git
cd frontend
bun install --frozen-lockfile
bun run build --filter=ppanel-admin-web
```

使用 Nginx 容器托管构建产物：

```bash
docker run -d --name ppanel-admin-web --restart unless-stopped \
  -p 3001:80 \
  -v "$(pwd)/apps/admin/dist:/usr/share/nginx/html:ro" \
  nginx:alpine
```

在生产环境中，请使用反向代理为管理端配置 HTTPS，并限制管理端的访问来源。
