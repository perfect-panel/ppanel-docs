# 使用 Vercel 部署管理端

管理端位于当前 [frontend](https://github.com/perfect-panel/frontend) 仓库的
`apps/admin`。在 Vercel 导入仓库时选择该目录作为 Root Directory，并设置：

```text
VITE_API_BASE_URL=https://api.example.com
VITE_CDN_URL=https://cdn.example.com
```

可使用官方仓库的预填部署链接：

[![在 Vercel 上部署](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https%3A%2F%2Fgithub.com%2Fperfect-panel%2Ffrontend&root-directory=apps%2Fadmin&project-name=ppanel-admin-web)

环境变量会在构建时写入静态文件。修改后需触发新的部署；不要继续使用旧
`ppanel-web` 的 Vercel 链接。
