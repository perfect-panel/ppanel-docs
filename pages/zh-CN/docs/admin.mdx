# 部署指南

## 环境变量

在部署之前，请确保在部署命令中直接设置以下环境变量，以配置应用程序：

```env
# 应用程序的默认语言
NEXT_PUBLIC_DEFAULT_LANGUAGE=<您的默认语言 zh-CN | en-US>

# 网站 URL 和 API 端点
NEXT_PUBLIC_SITE_URL=<您的网站 URL>
NEXT_PUBLIC_API_URL=<您的 API URL>

# 支持或咨询的联系邮箱
NEXT_PUBLIC_EMAIL=<您的联系邮箱>

# 社区和社交媒体链接
NEXT_PUBLIC_TELEGRAM_LINK=<您的 Telegram 链接>
NEXT_PUBLIC_TWITTER_LINK=<您的 Twitter 链接>
NEXT_PUBLIC_DISCORD_LINK=<您的 Discord 链接>
NEXT_PUBLIC_INSTAGRAM_LINK=<您的 Instagram 链接>
NEXT_PUBLIC_LINKEDIN_LINK=<您的 LinkedIn 链接>
NEXT_PUBLIC_FACEBOOK_LINK=<您的 Facebook 链接>
NEXT_PUBLIC_GITHUB_LINK=<您的 GitHub 链接>

# 默认用户凭据（用于初始设置或测试）
NEXT_PUBLIC_DEFAULT_USER_EMAIL=<默认用户邮箱>
NEXT_PUBLIC_DEFAULT_USER_PASSWORD=<默认用户密码>
```

参考[示例环境文件](https://raw.githubusercontent.com/perfect-panel/ppanel-admin-web/refs/heads/main/.env.development)以获取结构化的参考。

## 在 Vercel 上部署

点击下方按钮快速在 Vercel 上部署：

[![在 Vercel 上部署](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https%3A%2F%2Fgithub.com%2Fperfect-panel%2Fppanel-admin-web&env=NEXT_PUBLIC_API_URL&envDescription=环境变量说明&envLink=https%3A%2F%2Fraw.githubusercontent.com%2Fperfect-panel%2Fppanel-admin-web%2Frefs%2Fheads%2Fmain%2F.env.development&project-name=ppanel-admin-web&repository-name=ppanel-admin-web)

### Vercel 部署步骤

1. 登录到您的 [Vercel 账户](https://vercel.com/)，如果没有则创建一个。
2. 点击上方的 **在 Vercel 上部署** 按钮。
3. 在 Vercel 控制台中，设置必需的环境变量，例如 `NEXT_PUBLIC_API_URL`。
4. 点击 **Deploy** 开始部署。
5. 部署完成后，可在 Vercel 控制台中查看实际网址。

## 使用 Docker 部署


### 运行 Docker 容器

1. **使用最新版本**：

   ```sh
   docker run -d -p 3000:3000 \
     --env NEXT_PUBLIC_DEFAULT_LANGUAGE=en-US \
     --env NEXT_PUBLIC_SITE_URL=https://example.com \
     --env NEXT_PUBLIC_API_URL=https://api.example.com \
     --env NEXT_PUBLIC_EMAIL=contact@example.com \
     --env NEXT_PUBLIC_GITHUB_LINK=https://github.com/example/repository \
     --env NEXT_PUBLIC_DEFAULT_USER_EMAIL=user@example.com \
     --env NEXT_PUBLIC_DEFAULT_USER_PASSWORD=password123 \
     --name ppanel-admin-web ppanel/ppanel-admin-web:latest
   ```

2. **使用 Beta 版本**：

   ```sh
   docker run -d -p 3000:3000 \
     --env NEXT_PUBLIC_DEFAULT_LANGUAGE=en-US \
     --env NEXT_PUBLIC_SITE_URL=https://example.com \
     --env NEXT_PUBLIC_API_URL=https://api.example.com \
     --env NEXT_PUBLIC_EMAIL=contact@example.com \
     --env NEXT_PUBLIC_GITHUB_LINK=https://github.com/example/repository \
     --env NEXT_PUBLIC_DEFAULT_USER_EMAIL=user@example.com \
     --env NEXT_PUBLIC_DEFAULT_USER_PASSWORD=password123 \
     --name ppanel-admin-web-beta ppanel/ppanel-admin-web:beta
   ```

### 使用 Docker Compose 部署

3. **使用 Docker Compose（最新版本）**：
   创建 `docker-compose.yml` 文件，并配置如下：

   ```yaml
   version: '3'

   services:
     ppanel-admin-web:
       image: ppanel/ppanel-admin-web:latest
       ports:
         - '3000:3000'
       environment:
         NEXT_PUBLIC_DEFAULT_LANGUAGE: en-US
         NEXT_PUBLIC_SITE_URL: https://example.com
         NEXT_PUBLIC_API_URL: https://api.example.com
         NEXT_PUBLIC_EMAIL: contact@example.com
         NEXT_PUBLIC_GITHUB_LINK: https://github.com/example/repository
         NEXT_PUBLIC_DEFAULT_USER_EMAIL: user@example.com
         NEXT_PUBLIC_DEFAULT_USER_PASSWORD: password123
   ```

   启动服务：

   ```sh
   docker-compose up -d
   ```

4. **使用 Docker Compose（Beta 版本）**：
   创建 `docker-compose-beta.yml` 文件，并配置如下：

   ```yaml
   version: '3'

   services:
     ppanel-admin-web-beta:
       image: ppanel/ppanel-admin-web:beta
       ports:
         - '3001:3000'
       environment:
         NEXT_PUBLIC_DEFAULT_LANGUAGE: en-US
         NEXT_PUBLIC_SITE_URL: https://example.com
         NEXT_PUBLIC_API_URL: https://api.example.com
         NEXT_PUBLIC_EMAIL: contact@example.com
         NEXT_PUBLIC_GITHUB_LINK: https://github.com/example/repository
         NEXT_PUBLIC_DEFAULT_USER_EMAIL: user@example.com
         NEXT_PUBLIC_DEFAULT_USER_PASSWORD: password123
   ```

   启动 Beta 服务：

   ```sh
   docker-compose -f docker-compose-beta.yml up -d
   ```
