# 使用 Docker 托管用户端

用户端是 Vite 静态站点。先按[环境变量说明](./config)配置
`apps/user/.env.production`，然后构建：

```bash
git clone https://github.com/perfect-panel/frontend.git
cd frontend
bun install --frozen-lockfile
bun run build --filter=ppanel-user-web
```

使用 Nginx 容器托管静态文件：

```bash
docker run -d --name ppanel-user-web --restart unless-stopped \
  -p 3002:80 \
  -v "$(pwd)/apps/user/dist:/usr/share/nginx/html:ro" \
  nginx:alpine
```

为公网域名配置 HTTPS；`VITE_API_BASE_URL` 必须指向浏览器可访问的后端 API 地址。
