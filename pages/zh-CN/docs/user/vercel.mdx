# 使用 Vercel 部署用户端

用户端位于当前 [frontend](https://github.com/perfect-panel/frontend) 仓库的
`apps/user`。在 Vercel 中选择该目录为 Root Directory，并至少设置：

```text
VITE_API_BASE_URL=https://api.example.com
```

[![在 Vercel 上部署](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https%3A%2F%2Fgithub.com%2Fperfect-panel%2Ffrontend&root-directory=apps%2Fuser&project-name=ppanel-user-web)

修改环境变量后必须重新部署。旧 `ppanel-web` 部署链接已不适用于当前前端仓库。
