# 从源码部署用户端

用户端当前是 Vite 应用，不再提供旧版 `ppanel-web` 的固定 `v1.0.0` 下载包。

```bash
git clone https://github.com/perfect-panel/frontend.git
cd frontend
cp apps/user/.env.example apps/user/.env.production
# 编辑 apps/user/.env.production，至少设置 VITE_API_BASE_URL
bun install --frozen-lockfile
bun run build --filter=ppanel-user-web
```

将 `apps/user/dist/` 部署到静态 Web 服务器、对象存储或 CDN。Node.js/Bun 仅用于
构建阶段。

完整方案见
[前端部署指南](https://github.com/perfect-panel/frontend/blob/main/docs/zh/guide/separation/frontend.md)。
