# 部署指南

## 环境变量

在部署前，请确保在部署命令中直接设置以下环境变量以配置应用程序：

```bash
# 应用的默认语言
NEXT_PUBLIC_DEFAULT_LANGUAGE=<您的默认语言 zh-CN | en-US>

# 站点 URL 和 API 端点
NEXT_PUBLIC_SITE_URL=<您的站点 URL>
NEXT_PUBLIC_API_URL=<您的 API URL>

# 支持或咨询的联系邮箱
NEXT_PUBLIC_EMAIL=<您的联系邮箱>

# 社区和社交媒体链接
NEXT_PUBLIC_TELEGRAM_LINK=<您的 Telegram 链接>
NEXT_PUBLIC_TWITTER_LINK=<您的 Twitter 链接>
NEXT_PUBLIC_DISCORD_LINK=<您的 Discord 链接>
NEXT_PUBLIC_INSTAGRAM_LINK=<您的 Instagram 链接>
NEXT_PUBLIC_LINKEDIN_LINK=<您的 LinkedIn 链接>
NEXT_PUBLIC_FACEBOOK_LINK=<您的 Facebook 链接>
NEXT_PUBLIC_GITHUB_LINK=<您的 GitHub 链接>

# 默认用户凭证（用于初始设置或测试目的）
NEXT_PUBLIC_DEFAULT_USER_EMAIL=<您的默认用户邮箱>
NEXT_PUBLIC_DEFAULT_USER_PASSWORD=<您的默认用户密码>
```

可以参考[示例环境文件](https://raw.githubusercontent.com/perfect-panel/ppanel-user-web/refs/heads/main/.env.development)来查看结构化的参考文件。

## 在 Vercel 上部署

要快速在 Vercel 上部署，请点击下方按钮：

[![使用 Vercel 部署](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https%3A%2F%2Fgithub.com%2Fperfect-panel%2Fppanel-user-web&env=NEXT_PUBLIC_API_URL&envDescription=环境变量描述&envLink=https%3A%2F%2Fraw.githubusercontent.com%2Fperfect-panel%2Fppanel-user-web%2Frefs%2Fheads%2Fmain%2F.env.development&project-name=ppanel-user-web&repository-name=ppanel-user-web)

### Vercel 部署步骤

1. 登录到您的 [Vercel 账户](https://vercel.com/) 或创建一个账户。
2. 点击上方的 **使用 Vercel 部署** 按钮。
3. 在 Vercel 控制台中，设置所需的环境变量，如 `NEXT_PUBLIC_API_URL`。
4. 点击 **Deploy** 开始部署。
5. 部署完成后，可以在 Vercel 控制台中查看实时 URL。

## 使用 Docker 部署

### 安装 Docker

运行以下命令安装 Docker：

```sh
curl -fsSL https://get.docker.com | bash -s docker
```

### 运行 Docker 容器

1. **使用最新版本**：

   ```sh
   docker run -d -p 3000:3000 \
     --env NEXT_PUBLIC_DEFAULT_LANGUAGE=en-US \
     --env NEXT_PUBLIC_SITE_URL=https://example.com \
     --env NEXT_PUBLIC_API_URL=https://api.example.com \
     --env NEXT_PUBLIC_EMAIL=contact@example.com \
     --env NEXT_PUBLIC_TELEGRAM_LINK=https://t.me/example \
     --env NEXT_PUBLIC_TWITTER_LINK=https://twitter.com/example \
     --env NEXT_PUBLIC_DISCORD_LINK=https://discord.com/example \
     --env NEXT_PUBLIC_INSTAGRAM_LINK=https://instagram.com/example \
     --env NEXT_PUBLIC_LINKEDIN_LINK=https://linkedin.com/example \
     --env NEXT_PUBLIC_FACEBOOK_LINK=https://facebook.com/example \
     --env NEXT_PUBLIC_GITHUB_LINK=https://github.com/example/repository \
     --env NEXT_PUBLIC_DEFAULT_USER_EMAIL=user@example.com \
     --env NEXT_PUBLIC_DEFAULT_USER_PASSWORD=password123 \
     --name ppanel-user-web ppanel/ppanel-user-web:latest
   ```

2. **使用 Beta 版本**：


### 使用 Docker Compose

3. **使用 Docker Compose（最新版本）**：
   创建一个 `docker-compose.yml` 文件并配置如下：

   ```yaml
   version: '3'

   services:
     ppanel-user-web:
       image: ppanel/ppanel-user-web:latest
       ports:
         - '3000:3000'
       environment:
         NEXT_PUBLIC_DEFAULT_LANGUAGE: en-US
         NEXT_PUBLIC_SITE_URL: https://example.com
         NEXT_PUBLIC_API_URL: https://api.example.com
         NEXT_PUBLIC_EMAIL: contact@example.com
         NEXT_PUBLIC_TELEGRAM_LINK: https://t.me/example
         NEXT_PUBLIC_TWITTER_LINK: https://twitter.com/example
         NEXT_PUBLIC_DISCORD_LINK: https://discord.com/example
         NEXT_PUBLIC_INSTAGRAM_LINK: https://instagram.com/example
         NEXT_PUBLIC_LINKEDIN_LINK: https://linkedin.com/example
         NEXT_PUBLIC_FACEBOOK_LINK: https://facebook.com/example
         NEXT_PUBLIC_GITHUB_LINK: https://github.com/example/repository
         NEXT_PUBLIC_DEFAULT_USER_EMAIL: user@example.com
         NEXT_PUBLIC_DEFAULT_USER_PASSWORD: password123
   ```

   启动服务：

   ```sh
   docker compose up -d
   ```

4. **使用 Docker Compose（Beta 版本）**：
   创建一个 `docker-compose-beta.yml` 文件并配置如下：

   ```yaml
   version: '3'

   services:

       image: ppanel/ppanel-user-web:beta
       ports:
         - '3001:3000'
       environment:
         NEXT_PUBLIC_DEFAULT_LANGUAGE: en-US

         NEXT_PUBLIC_TWITTER_LINK: https://twitter.com/example
         NEXT_PUBLIC_DISCORD_LINK: https://discord.com/example
         NEXT_PUBLIC_INSTAGRAM_LINK: https://instagram.com/example
         NEXT_PUBLIC_LINKEDIN_LINK: https://linkedin.com/example
         NEXT_PUBLIC_FACEBOOK_LINK: https://facebook.com/example
         NEXT_PUBLIC_GITHUB_LINK: https://github.com/example/repository
         NEXT_PUBLIC_DEFAULT_USER_EMAIL: user@example.com
   ```

   启动 Beta 服务：

   ```sh
   docker compose -f docker-compose-beta.yml up -d
   ```
