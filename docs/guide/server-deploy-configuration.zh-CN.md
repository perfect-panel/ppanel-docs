## 运行环境要求

| 组件         | 要求                                               |
| ------------ | -------------------------------------------------- |
| 服务器配置   | 最低：1 核 CPU，2GB 内存；推荐：2 核 CPU，4GB 内存 |
| MySQL        | 5.7 及以上（推荐 MySQL 8）                         |
| Redis        | 6 及以上                                           |
| NGINX/Apache | 任意版本                                           |
