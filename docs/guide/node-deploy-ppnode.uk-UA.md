# PPnode

PPanel — це чистий, професійний та досконалий інструмент для проксі-панелей з відкритим кодом, який прагне стати вашим ідеальним вибором для навчання та практики.

## Функціональні можливості

- Назавжди відкритий код та безкоштовний.
- Підтримує різні протоколи, включаючи V2ray, Trojan та Shadowsocks.
- Підтримує нові функції, такі як Vless та XTLS.
- Підтримує інтеграцію кількох вузлів в одному екземплярі без повторного запуску.
- Можливість обмеження онлайн IP.
- Підтримує обмеження кількості TCP з'єднань.
- Підтримує обмеження швидкості на рівні вузлів та користувачів.
- Просте та зрозуміле налаштування.
- Автоматичний перезапуск екземпляра після зміни конфігурації.
- Підтримує різні ядра, легко розширює функціональність.
- Підтримує умовну компіляцію, включаючи лише необхідні ядра.
