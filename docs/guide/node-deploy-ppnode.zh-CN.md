# PPnode

PPanel 是一个纯粹、专业、完美的开源代理面板工具，致力于成为您学习和实践的理想选择。

## 功能特点

- 永久开源且永久免费。  
- 支持多种协议，包括 V2ray、Trojan 和 Shadowsocks。  
- 支持 Vless 和 XTLS 等新特性。  
- 支持单实例多节点集成，无需重复启动。  
- 可限制在线 IP。  
- 支持限制 TCP 连接数量。  
- 支持节点端口和用户级别的带宽限速。  
- 配置简单清晰。  
- 配置更改后可自动重启实例。  
- 支持多种内核，易于扩展功能。  
- 支持条件编译，仅包含所需的内核。
