# PPnode

PPanel ist ein reines, professionelles und perfektes Open-Source-Proxy-Panel-Tool, das darauf abzielt, Ihre ideale Wahl für Lernen und Praxis zu sein.

## Funktionen

- Dauerhaft Open Source und kostenlos.
- Unterstützt verschiedene Protokolle, einschließlich V2ray, Trojan und Shadowsocks.
- Unterstützt neue Funktionen wie Vless und XTLS.
- Unterstützt die Integration mehrerer Knoten in einer einzelnen Instanz, ohne dass ein mehrfacher Start erforderlich ist.
- Kann die Anzahl der Online-IP-Adressen begrenzen.
- Unterstützt die Begrenzung der Anzahl der TCP-Verbindungen.
- Unterstützt Bandbreitenbegrenzung auf Knotenport- und Benutzerebene.
- Einfache und klare Konfiguration.
- Startet die Instanz automatisch neu, nachdem die Konfiguration geändert wurde.
- Unterstützt verschiedene Kerne, die eine einfache Erweiterung der Funktionen ermöglichen.
- Unterstützt bedingte Kompilierung, die nur die benötigten Kerne enthält.
